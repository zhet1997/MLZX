# 自评自查判定规则

> 本文件是 `core/self_check.py` 中判定逻辑的人类可读说明。运行时不加载此文件。

## 1. 计分

| 选项 | 分值 |
|------|------|
| A    | 0    |
| B    | 1    |
| C    | 2    |

## 2. 维度与题号

| 维度 | 题号       | 题数 | 满分(raw) |
|------|------------|------|-----------|
| 立意 | Q1–Q4      | 4    | 8         |
| 结构 | Q5–Q8      | 4    | 8         |
| 论证 | Q9–Q14     | 6    | 12        |
| 语言 | Q15–Q16    | 2    | 4         |

## 3. 维度得分

- **raw** = 该维度所有题目分值之和
- **norm** = raw / (2 × 题数)，归一化到 [0, 1]
- **ccount** = 该维度选 C 的题目数量

## 4. 红灯规则

- **ideation_red** = Q1 选 C **或** Q2 选 C
- **pool**（红灯池）= ccount ≥ 1 的所有维度

## 5. 最需改进维度判定

按以下优先级依次判定，首个满足的规则即为最终结果：

### 规则 1：立意红灯
若 ideation_red = true → **final_dimension = 立意**（不比较其他维度）

### 规则 2：全选 A 特殊情况
若所有 16 题均选 A（所有 raw = 0）→ **final_dimension = 论证**

### 规则 3：红灯池非空
在 pool 中的维度里选 norm 最大者：
- 并列 → 选 ccount 最大者
- 仍并列 → 按固定优先级：论证 > 结构 > 语言

### 规则 4：红灯池为空
在全部四维度中选 norm 最大者：
- 并列 → 选 ccount 最大者
- 仍并列 → 按固定优先级：立意 > 论证 > 结构 > 语言

## 6. rationale 生成

- 只引用作答者的选择，不评价原文
- 优先列出 final_dimension 对应维度中所有选 C 的题号 + 关键词
- 其次列出选 B 的题号 + 关键词（可选）
- 若触发红灯，明确说明触发条款（Q1=C / Q2=C / 两者皆是）
